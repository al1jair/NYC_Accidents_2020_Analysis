# NYC Accidents 2020 Analysis

### Project Overview

This project analyzes the NYC Accidents 2020 dataset provided by Maven Analytics, to identify patterns in crash frequency by time, location, and contributing factors. The analysis was completed entirely in R using tidyverse, lubridate, and ggplot2, and documented in an R Markdown report.
